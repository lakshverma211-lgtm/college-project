CSS Selectors

Universal Selector (*)

It applies styles to every element on the page, ensuring consistency in font and spacing.

ID Selector (#)

The top heading uses an ID selector #page-title for a unique style and alignment.

Class Selector (.)

These boxes use a .card class for shared border, shadow, and padding styles.

Descendant Selector

The selector .card p styles only paragraphs inside the card, not elsewhere.

Group Selector

The headings h2 and h3 share color and alignment using a group selector.
