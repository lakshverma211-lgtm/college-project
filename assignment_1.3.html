Function of <img> Tag in HTML

the < img> ; Tag in HTML is used to display images on a webpage. it is an empty (self-closing) tag, which means it does not have a closing tag. instead, all the information about the image is provided through its attributes.

Example :







Important Attribute :

  • src - specifies the path (URL or file location) of the image. This attribute is mandatory, as it tells the browser where to find the image.
  • alt - Provides alternative text for the image if it cannot be displayed. It is important for accessibility(screen readers) and SEO.
  • width and height- optional attribute that define the size of the image in pixels.

Explanation :

using the src attribute ensures the image is displayed correctly, while thealt attribute improves accessibility and provides information when the image cannot be loaded. the width and height attributes help control the image dimentions for proper layout on the webpage.
